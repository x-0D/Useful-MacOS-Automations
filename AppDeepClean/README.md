Установка:
1. Установить двойным кликом
2. CMD+Shift+G перейти в расположение `~/.Trash/`
3. Показать строку пути, если еще не включена Opt+CMD+P
4. ПКМ по директории корзины в пути, Службы -> Настройка действий папки...
5. Разрешить действия, добавить для корзины AppDeepClean workflow
6. Готово

Вдохновлено https://github.com/alienator88/Pearcleaner

ИМХО - целое приложение для этой задачи слишком много, но без Pearcleaner не существовало бы этой автоматизации.
